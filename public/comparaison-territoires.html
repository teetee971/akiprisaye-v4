🗺️ Comparaison Territoriale des Prix

Comparaison basée EXCLUSIVEMENT sur des prix réels observés

⚠️ Avertissement méthodologique

  • Aucune estimation : Les prix affichés proviennent UNIQUEMENT de tickets de caisse réels
  • Aucune moyenne fictive : Si un territoire manque de données, cela sera clairement indiqué
  • Aucun lissage statistique : Les calculs (min, max, médiane) sont basés sur les observations brutes
  • Transparence totale : Le nombre d'observations est toujours affiché

🔍 Critères de comparaison

Recherchez un produit spécifique par son nom
Maintenez Ctrl (Cmd sur Mac) pour sélectionner plusieurs territoires
Laissez vide pour inclure toutes les observations disponibles
Données réelles – observatoire citoyen

📊 À propos de cette comparaison

  • Ce qui est comparé : Prix réels observés sur des tickets de caisse vérifiés
  • Ce qui n'est PAS comparé : Estimations, prix moyens fictifs, données extrapolées
  • Médiane : Calculée uniquement si ≥ 3 observations (sinon affichée comme "Insuffisant")
  • Limites : Certains territoires peuvent manquer de données car la collecte est citoyenne
  • Enseignes : La liste des enseignes observées est affichée pour traçabilité
