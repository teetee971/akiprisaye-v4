Exemple d'utilisation dynamique

// JavaScript
function getNutriScoreIcon(score) {
  const normalizedScore = score.toUpperCase();
  if (!['A', 'B', 'C', 'D', 'E'].includes(normalizedScore)) {
    return null;
  }
  return `/public/assets/nutriscore-${normalizedScore.toLowerCase()}.svg`;
}

// Utilisation
const productScore = 'B';
const iconPath = getNutriScoreIcon(productScore);
document.getElementById('nutriscore').src = iconPath;

⚙️ Informations Techniques

Optimisation

  • ✅ Format SVG pour une taille de fichier minimale
  • ✅ Rendu net à toutes les résolutions (vectoriel)
  • ✅ Balises <title> pour l'accessibilité
  • ✅ Taille moyenne : ~2KB par icône

Accessibilité

  • Toujours inclure un attribut alt descriptif
  • Utiliser aria-label pour les éléments interactifs
  • Fournir un contexte textuel si nécessaire

Exemple accessible

<img 
  src="public/assets/nutriscore-a.svg" 
  alt="Produit avec Nutri-Score A - Très bonne qualité nutritionnelle" 
  width="200" 
  height="100"
