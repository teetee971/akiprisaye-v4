Applies to: Observed price data, citizen contributions

- **URL:** https://opendatacommons.org/licenses/odbl/1.0/

Applies to: Documentation, methodologies

- **File:** [CC-BY-4.0.txt](./CC-BY-4.0.txt)

## Usage

When using data from this repository, please:

1. Check the specific dataset license in its metadata
2. Provide appropriate attribution
3. Include a link to the license
4. Indicate if changes were made

## Example Attribution

```
Data from: Observatoire du Coût de la Vie (2026)
Dataset: Cost of Living - Martinique
License: ODbL-1.0
URL: https://github.com/teetee971/akiprisaye-web
Retrieved: 2026-01-02
```
