## FAQ Étendue - Abonnements

### 1. Général

**Q: Pourquoi proposer des abonnements si c'est un service public ?**

R: Le service de base reste gratuit pour tous. Les abonnements permettent de :
- Financer l'infrastructure technique (serveurs, stockage, calcul)
- Maintenir et améliorer les données
- Garantir l'indépendance (pas de publicité, pas de revente de données)
- Assurer la pérennité à long terme

**Q: Puis-je annuler mon abonnement à tout moment ?**

R: Oui, absolument. L'annulation peut se faire à tout moment depuis votre espace personnel. Aucune pénalité, aucun engagement sur la durée. Vous conservez l'accès jusqu'à la fin de la période payée.

**Q: Y a-t-il des réductions pour les étudiants ou les associations ?**

R: Actuellement, le tarif "Pro" à 9,99€/mois est déjà pensé pour être accessible aux associations et chercheurs. Pour les institutions publiques et universités, des conventions spécifiques peuvent être établies.

### 2. Comparaison des niveaux

**Q: Quelle est la différence entre Gratuit et Citoyen+ ?**

R: 
- **Gratuit** : Consultation des comparateurs de base, données actuelles, lecture seule
- **Citoyen+** : + Historique étendu (12 mois), alertes personnalisées, exports basiques, suivi de produits

**Q: Quelle est la différence entre Citoyen+ et Pro ?**

R:
- **Citoyen+** : Focus territoire unique, historique 12 mois, exports basiques
- **Pro** : Comparaisons multi-territoires, historique long (5 ans), exports CSV/JSON complets, agrégations avancées

**Q: Le niveau Institution donne-t-il accès à des données exclusives ?**

R: **Non**. Les données sont identiques pour tous. La licence institutionnelle donne accès à :
- Des formats d'export normalisés (INSEE, Eurostat)
- Une API pour l'automatisation
- Un support méthodologique
- Une documentation technique complète

Mais les données sources restent les mêmes pour tous.

### 3. Paiement et facturation

**Q: Le paiement est-il activé ?**

R: **Non, pas encore**. Pour l'instant, l'accès se fait sur demande ou par convention. Le système de paiement sera activé progressivement après validation complète de la plateforme.

**Q: Quels moyens de paiement seront acceptés ?**

R: (À préciser lors de l'activation)
- Carte bancaire
- Virement SEPA (pour institutions)
- Possibilité de conventions pour collectivités

**Q: Y a-t-il une TVA ?**

R: Oui, la TVA française s'applique (20%) pour les particuliers. Les structures institutionnelles avec TVA intracommunautaire peuvent être exonérées.

**Q: Puis-je payer annuellement ?**

R: Oui, un tarif annuel sera proposé avec une réduction (exemple : 10 mois au prix de 12).

### 4. Fonctionnalités et limites

**Q: Y a-t-il des limites d'export avec Citoyen+ ?**

R: Oui, les exports Citoyen+ sont limités à :
- 50 exports par mois
- Format CSV simple
- 1 territoire à la fois

Le niveau Pro offre des exports illimités et multi-territoires.

**Q: Puis-je partager mon compte avec d'autres personnes ?**

R: Non. Chaque compte est personnel et nominatif pour des raisons de traçabilité. Pour les équipes, le niveau Pro ou Institution est recommandé.

**Q: Les alertes fonctionnent-elles même avec le compte gratuit ?**

R: Non. Les alertes automatiques sont réservées aux niveaux Citoyen+ et supérieurs. Le compte gratuit permet de consulter manuellement les données.

### 5. Institutions et recherche

**Q: Comment obtenir une licence institutionnelle ?**

R: Contactez-nous via le formulaire dédié en précisant :
- Votre structure (université, collectivité, institut de recherche)
- L'usage prévu
- Le volume de données nécessaire
- Les besoins en API

Nous établirons une convention adaptée.

**Q: Les données sont-elles auditables ?**

R: Oui, totalement. La méthodologie de collecte, les sources et les transformations sont documentées et vérifiables. Les institutions peuvent demander un audit complet.

**Q: Y a-t-il un SLA (Service Level Agreement) ?**

R: Pour les licences institutionnelles, oui. Un SLA peut être défini dans la convention (disponibilité, temps de réponse, support).

### 6. Données et protection

**Q: Mes historiques de consultation sont-ils suivis ?**

R: Uniquement pour l'agrégation statistique anonyme (compteurs publics). Aucun tracking commercial ou publicitaire. Conformité RGPD totale.

**Q: Puis-je demander l'effacement de mes données ?**

R: Oui, conformément au RGPD. Vous pouvez demander l'effacement de votre compte et de toutes vos données personnelles à tout moment.

**Q: Les données de prix sont-elles en temps réel ?**

R: Elles sont actualisées quotidiennement. Ce n'est pas du "temps réel" strict mais un observatoire fiable des tendances de prix.

### 7. Gratification

**Q: À quoi servent les badges ?**

R: C'est une reconnaissance sobre et non-compétitive de votre usage du service. Ils n'ont aucune valeur marchande, aucun classement public. Ils servent uniquement à contextualiser votre profil.

**Q: Puis-je refuser les badges ?**

R: Oui. Vous pouvez masquer les badges dans vos paramètres. Ils restent une fonctionnalité optionnelle.

**Q: Les compteurs sont-ils publics ?**

R: Non. Seuls les compteurs globaux du service (nombre total d'utilisateurs, d'exports) sont publics. Vos statistiques personnelles restent privées.

### 8. Évolutions futures

**Q: Y aura-t-il de nouveaux niveaux d'abonnement ?**

R: Non, la structure à 4 niveaux (Gratuit / Citoyen+ / Pro / Institution) est figée. Elle couvre tous les cas d'usage identifiés.

**Q: De nouvelles fonctionnalités seront-elles ajoutées ?**

R: Oui, notamment :
- Modules comparateurs (énergie, transports)
- Assistant intelligent (FAQ automatisée)
- Exports enrichis
- API étendue

Les nouvelles fonctionnalités seront intégrées aux niveaux existants selon leur pertinence.

## Politique de Transparence

### Utilisation des fonds

Les contributions financières sont utilisées pour :
1. Infrastructure technique (70%) : serveurs, stockage, calcul
2. Maintenance et développement (20%) : améliorations, sécurité
3. Ouverture des données (10%) : documentation, support, formations

### Engagement

- Aucune publicité
- Aucune revente de données
- Aucun tracking commercial
- Transparence totale sur la méthodologie
- Conformité RGPD stricte
