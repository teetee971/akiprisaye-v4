# Documentation Abonnements - A KI PRI SA YÉ v1.6.1

## Vue d'ensemble

Ce document détaille le modèle économique définitif de A KI PRI SA YÉ, aligné sur les principes d'un observatoire citoyen indépendant.

## Objectif

Établir un modèle économique :
- **Crédible** : Tarifs justes et transparents
- **Institutionnellement défendable** : Compatible secteur public
- **Pérenne** : Assure la continuité du service
- **Sans dérive commerciale** : Pas de publicité, pas de revente de données

## Structure Tarifaire Finale

### 📖 Gratuit - 0€

**Public cible** : Tous les citoyens

**Inclus** :
- Comparateur citoyen DOM · ROM · COM
- Consultation des prix observés
- Données datées et sourcées
- Historique simple
- Lecture seule
- Sans publicité

**Limites** :
- Pas d'alertes automatiques
- Pas d'exports
- Pas d'historique étendu

**Philosophie** : L'accès de base aux données publiques est un droit.

---

### 🧑 Citoyen - 3,99€ / mois

**Public cible** : Citoyens engagés, soutiens du service

**Inclus tout le gratuit +** :
- Scan EAN illimité (jamais bloqué)
- OCR ingrédients (texte brut)
- Fiche produit enrichie
- Alertes locales de variation de prix
- Historique personnel local
- Signalement citoyen
- Comparaison temporelle basique

**N'inclut PAS** :
- Comparaisons temporelles multi-marques
- Historique long (> 12 mois)
- Exports CSV/JSON avancés
- API

**Philosophie** : Valeur immédiate. Le scan n'est jamais bloqué, aucune frustration post-scan.

**Tarif justifié par** :
- Coût serveur pour stockage historique
- Coût notification (emails, alertes)
- Maintenance infrastructure OCR

---

### 🧑‍💼 Professionnel - 19€ / mois

**Public cible** : Associations, artisans, journalistes, chercheurs indépendants

**Inclus tout Citoyen +** :
- Comparaisons temporelles multi-marques
- Historique long (12-36 mois)
- Séries temporelles longues (jusqu'à 5 ans)
- Exports CSV / JSON complets
- Agrégation territoriale
- Recherche EAN + historique
- Méthodologie détaillée
- Accès anticipé aux nouvelles fonctionnalités

**Limites** :
- Outil d'observation, pas de conseil
- Usage non commercial
- Pas d'API automatisée
- Pas de redistribution

**Philosophie** : Outil d'analyse territoriale pour acteurs de l'information et de la recherche.

**Tarif justifié par** :
- Coûts de calcul agrégations avancées
- Stockage séries temporelles longues
- Support technique étendu

---

### 🏛️ Institution - Sur devis (Convention annuelle)

**Public cible** : Secteur public, universités, instituts de recherche

**Inclus tout Pro +** :
- Accès API open-data
- Exports normalisés (INSEE, Eurostat)
- Séries consolidées multi-services
- Accès indices globaux
- Documentation technique officielle
- Support méthodologique dédié
- Possibilité dépôt pilote
- Auditabilité complète

**Garanties** :
- Données observées uniquement
- Traçabilité totale
- Conformité réglementaire
- SLA défini par convention

**Philosophie** : Cadre officiel pour partenariats institutionnels.

**Tarif sur devis car** :
- Besoins variables selon la structure
- Volumes de données différents
- Intégrations spécifiques possibles
- Support dédié

## Système de Gratification v1.6.1

### Objectif

Reconnaissance sobre, non-compétitive de l'usage du service.

### Principes

- ⛔ **Pas de gamification toxique**
- ⛔ **Pas de classement public**
- ⛔ **Pas de récompense financière**
- ✅ **Reconnaissance purement informative**
- ✅ **Indications privées uniquement**

### Badges Disponibles

#### 🏅 Badge Contributeur citoyen
- **Critère** : Participation aux signalements et améliorations
- **Niveau** : Tous niveaux
- **Affichage** : Profil utilisateur (privé)

#### 👁️ Badge Veilleur de prix
- **Critère** : 30+ scans actifs (usage régulier)
- **Niveau** : Tous niveaux
- **Affichage** : Profil utilisateur (privé)

#### 🏛️ Mention Partenaire institutionnel
- **Critère** : Convention active
- **Niveau** : Institution
- **Affichage** : Profil utilisateur (privé)

### Compteurs d'Usage

**Affichés sur le profil utilisateur** :

1. **Produits scannés** : Nombre de scans EAN réalisés
2. **Contributions utiles** : Signalements et améliorations validés
3. **Téléchargements** : Nombre d'exports réalisés
4. **Jours actifs** : Nombre de jours d'utilisation

**Message de contribution** :
"💚 Votre contribution améliore la transparence locale"

**Caractéristiques** :
- Compteurs agrégés
- Pas de détail par action
- Pas de comparaison entre utilisateurs
- Affichage optionnel (peut être masqué)

### Statut Utilisateur (Lecture Seule)

Indication simple du niveau d'accès :
- "Utilisateur public"
- "Citoyen+ actif"
- "Utilisateur Pro"
- "Partenaire institutionnel"

**Usage** : Contextualiser l'accès aux fonctionnalités uniquement.
