# Guide Comparateur Fret Maritime & Colis

## 📦 Présentation

Le **Comparateur Fret Maritime & Colis** est le **premier comparateur en France** à offrir une transparence totale sur les coûts d'envoi vers les Outre-mer.

### Pourquoi ce comparateur ?

Le fret maritime représente **80% des importations DOM-TOM** et est le **principal facteur de vie chère** en Outre-mer selon :
- Rapports Sénat 2024-2025
- Commission d'enquête Assemblée Nationale sur la vie chère
- 68 mesures CIOM (Comité Interministériel des Outre-Mer)

### Problèmes identifiés

1. **Manque de transparence totale** sur les frais
2. **Délais annoncés ≠ délais réels**
3. **Frais cachés** (octroi de mer, taxes douanières)
4. **Peu d'alternatives connues**

---

1. ✅ **Transparence totale sur l'octroi de mer**

## 🚀 Comment utiliser le comparateur ?

### 1. Simuler un envoi

1. **Origine** : Sélectionnez la ville de départ (Paris, Marseille, Lyon, etc.)
2. **Destination** : Choisissez le territoire d'arrivée (GP, MQ, GY, RE, YT, etc.)
3. **Poids** : Indiquez le poids de votre colis (en kg)
4. **Dimensions** : Longueur, largeur, hauteur (en cm)
5. **Type de colis** :
   - Standard
   - Fragile
   - Valeur déclarée (avec assurance)
6. **Urgence** :
   - Standard
   - Express
   - Urgent
7. Cliquez sur **"Comparer les transporteurs"**

### 2. Comprendre les résultats

Le comparateur affiche :

- **Prix minimum** : Le transporteur le moins cher
- **Prix moyen** : Moyenne des prix
- **Prix maximum** : Le transporteur le plus cher
- **Écart de prix** : Différence en % entre min et max

Pour chaque transporteur :
- **Prix total TTC** (incluant octroi de mer)
- **Détail des frais** (prix de base, manutention, assurance, octroi)
- **Score de fiabilité** (sur 5 étoiles)
- **Délai réel moyen** (basé sur contributions)
- **Taux de ponctualité**
- **Incidents signalés**

### 3. Badges et indicateurs

- 🏆 **"Le moins cher"** : Prix le plus bas
- ✓ **"Meilleur rapport qualité/prix"** : Bon prix + fiabilité élevée
- ⭐ **Score de 0 à 5** : Fiabilité basée sur les avis

## 💰 Comprendre l'octroi de mer

L'**octroi de mer** est une taxe spécifique aux territoires d'Outre-mer appliquée sur les marchandises importées.

### Taux par territoire

| Territoire | Taux octroi de mer |
|------------|-------------------|
| Guadeloupe (GP) | 2.5% |
| Martinique (MQ) | 2.5% |
| Guyane (GF) | 5.0% |
| La Réunion (RE) | 2.5% |
| Mayotte (YT) | 3.0% |
| Saint-Martin (MF) | 2.0% |
| Saint-Barthélemy (BL) | 2.0% |
| Saint-Pierre-et-Miquelon (PM) | 2.0% |
| Wallis-et-Futuna (WF) | 2.5% |
| Polynésie française (PF) | 3.0% |
| Nouvelle-Calédonie (NC) | 2.5% |

### Calcul

```
Octroi de mer = Prix de base × Taux
```

**Exemple** : Pour un colis à 100€ vers la Guadeloupe (2.5%)
- Octroi de mer = 100€ × 0.025 = 2.50€

## 🤝 Comment contribuer ?

Vos contributions aident à améliorer la transparence pour tous !

### Prérequis

- Avoir envoyé un colis vers l'Outre-mer
- Avoir la facture (optionnel mais recommandé pour vérification)
- Connaître la date de réception

### Étapes

1. Cliquez sur **"Contribuer"**
2. **Uploadez votre facture** (format PDF ou image)
3. Le système extrait automatiquement :
   - Transporteur
   - Origine / Destination
   - Poids
   - Prix payé
   - Frais détaillés
4. **Vérifiez les données extraites** et corrigez si nécessaire
5. **Ajoutez la date de réception**
6. **Notez votre expérience** (1-5 étoiles)
7. **Commentaire** (optionnel)
8. Choisissez **anonyme** ou **signé**
9. **Validez**

### Contribution vérifiée ✅

- Avec facture = **Contribution vérifiée** ✅
- Sans facture = **À vérifier** ⏳
- Modération avant publication

## 🔔 Comment créer une alerte ?

Les alertes vous notifient des changements importants.

### Types d'alertes

1. **Alerte retard** : "Mon colis n'est toujours pas arrivé après X jours"
2. **Alerte baisse prix** : "Prix vers [destination] descend sous X€"
3. **Nouveau transporteur** : "Nouvelle option disponible pour [trajet]"
4. **Alerte problème** : "Incidents signalés avec [transporteur]"

### Créer une alerte

1. Cliquez sur **"Créer une alerte"**
2. Choisissez le **type d'alerte**
3. Configurez les **conditions** :
   - Route (origine → destination)
   - Prix maximum (pour alerte prix)
   - Délai maximum (pour alerte retard)
   - Transporteur spécifique (optionnel)
4. Choisissez le **mode de notification** :
   - Email
   - Push (PWA)
   - Les deux
5. **Validez**

### Gérer vos alertes

- Liste des alertes actives
- Modifier / Supprimer une alerte
- Historique des alertes déclenchées

## ❓ FAQ

### Le comparateur est-il gratuit ?

Oui, 100% gratuit. Aucune commission, aucun lien d'affiliation.

### Les prix sont-ils à jour ?

Les prix sont mis à jour quotidiennement à partir des sources officielles et des contributions citoyennes.

### Puis-je faire confiance aux données ?

Toutes les données sont sourcées et datées. Les contributions avec facture sont marquées "Vérifiées ✅".

### Comment sont calculés les délais réels ?

Les délais réels sont calculés à partir des dates d'envoi et de réception signalées par les contributeurs.

### Qu'est-ce qu'un frais caché ?

Un frais caché est un coût non annoncé au moment du devis mais facturé par la suite.

### Comment signaler un problème ?

Utilisez le formulaire de contribution pour signaler un incident avec un transporteur.
