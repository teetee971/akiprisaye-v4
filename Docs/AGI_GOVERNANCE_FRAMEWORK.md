# AGI Governance Framework

## Purpose
This document defines the governance framework for supervised Artificial General Intelligence (AGI)
within the A KI PRI SA YÉ platform.

## Principles
- Human-in-the-loop decision making
- Institutional oversight
- Transparency and auditability
- Ethical and proportional use
- Revocable autonomy

## Supervision Model
AGI systems operate with delegated autonomy under continuous human supervision,
with immediate override capability by authorized institutions.
